# 🎮 Auto HEN Installer HFW 4.93

Bienvenido al repositorio oficial del **Auto HEN Installer para HFW 4.93**, una herramienta esencial para la instalación de HEN (Homebrew Enabler) en consolas PlayStation 3. 

Esta versión cuenta con una interfaz gráfica estilizada y adaptada específicamente para brindar una mejor experiencia de usuario desde el navegador de la consola.

---

## ⚠️ Requisitos Previos

Antes de ejecutar este instalador en tu consola, es estrictamente necesario cumplir con lo siguiente:
- Tener una consola PS3 funcional.
- **Tener instalado el Firmware Híbrido (HFW) versión 4.93.** Si tienes una versión anterior o un OFW puro, el exploit fallará o mostrará un mensaje de advertencia.
- Conexión a internet activa en la consola.
